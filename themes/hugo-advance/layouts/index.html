Get Started

Nextflow can be installed on any POSIX compatible system (Linux, OS X, etc).

Simply follow these three steps.

Check
1
  • Java 8 or later is required
java -version


Version numbers 1.8.y_z and 8 identify the same Java release.

Install
2
  • One line installation
curl -s https://get.nextflow.io | bash

A file nextflow is created in the current directory.

Launch
3
  • Try a simple demo
./nextflow run hello


Start creating and launching workflows!
